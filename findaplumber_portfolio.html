Find a plumber

Web scrapper

Technologies: Python, Django

webapp → Github →

Find a plumber lists the information it can find about the plumbers in London.

The scrapper searches the website for the "plumbers" category and gets the links of each individual plumber from the list of results. It uses this information to go into each plumber individual page and get the contact information.

It was built as a test for a job interview.
